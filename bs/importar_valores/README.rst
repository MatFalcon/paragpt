********************
Importar valores PBP
********************

Cronjob para importar valores del PBP a la base de datos de ODOO.

Requisitos
==========

* Python >= 3.8
* Pip

Instalación
===========

1. Definir variables de configuración en ``config.ini``.

2. Ejecutar script de instalación:

   .. code-block:: bash

      $ bash install.sh
